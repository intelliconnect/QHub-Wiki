| Info                  | Description                              | 
| --------------------- | ------------------------------------ |
| WIKI_DOMAIN           | your domain eg. wiki.mydomain.com    |
| SERVER PUBLIC IP      | your public ip                       |  
| ANTHROPIC_API_KEY     | https://platform.claude.com/docs/en/api/overview                   | 
| GROQ_API_KEY          | https://console.groq.com/docs/quickstart  | 
| KB_REPO_NAME          | your github repository name ex: wiki-rag-kb  | 
| KB_REPO_OWNER         | Repository owner ex: mycompany| 
| KB_REPO_HTTP_URL      | your github HTTP knowledge base repository ex: https://github.com/mycompany/wiki-rag-kb.git | 
| KB_REPO_SSH_URL       | your github SSH knowledge base repository ex: git@github.com:mycompany/wiki-rag-kb.git|      
| KB_REPO_GITHUB_TOKEN  | auto-generated by github while creating FINE-GRAINED PERSONAL ACCESS TOKEN| 
| KB_GIT_COMMIT_NAME    | Your github username| 
| KB_GIT_COMMIT_EMAIL   | Your github email-id | 
| KB_GIT_BRANCH         | main| 
| INGEST_API_SECRET     | generated via OpenSSL                |   
| QDRANT_API_KEY        | generated via OpenSSL | 
| QDRANT_COLLECTION_NAME| qhubwikiqdrant | 
| WIKIJS_ADMIN_EMAIL    | your admin login id for WikiJS eg. jdoe@mycompany.com                               | 
| WIKIJS_ADMIN_PASSWORD | your password for WikiJS - alphanumeric - do not use $ sign in password                               |
